# CERBERUS Vigil Lab

A self-contained browser demonstrator for runtime independence budgeting.

## Open

Download `index.html` and open it in a modern browser. No server or build step is required.

## Demonstrated behaviors

- scenario-driven overlap growth;
- pessimistic FCOI upper-bound proxy;
- worst-class independence budget;
- immediate A3–A0 demotion;
- sequential dwell-gated promotion;
- fail-conservative Vigil fault;
- bounded sentinel campaign;
- monotonic Pilot shadow red-team update;
- bounded ground diagnostic recovery;
- event trace and CSV telemetry export.

## Interpretation boundary

The lab is an explanatory prototype. Its overlap values and thresholds are illustrative, not calibrated spacecraft quantities. It does not implement a minimal-cut-set engine, operational probabilistic FCOI, flight dynamics, cryptographic command handling, or hardware-in-the-loop sentinel validation.
